# 🧮 Calculadora Web

Uma calculadora simples desenvolvida com **HTML**, **CSS** e **JavaScript**, capaz de realizar operações matemáticas básicas de forma rápida e intuitiva.

## 📋 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🧹 Limpar visor
- ⌫ Apagar último caractere
- 📱 Layout responsivo
- 🎨 Interface moderna e amigável

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Estrutura do Projeto

```text
calculadora/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 💻 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/calculadora.git
```

2. Entre na pasta do projeto:

```bash
cd calculadora
```

3. Abra o arquivo `index.html` no navegador.

## 🛠️ Funcionalidades da Interface

| Botão | Função |
|--------|---------|
| 0-9 | Inserir números |
| + | Soma |
| - | Subtração |
| × | Multiplicação |
| ÷ | Divisão |
| C | Limpar visor |
| = | Calcular resultado |
